## spwn activity

View recent activity across worlds and agents

### Synopsis

Tails the system-wide activity log at ~/.spwn/activity.jsonl

```
spwn activity [flags]
```

### Options

```
  -a, --agent string   Filter by agent name
  -h, --help           help for activity
      --json           Output as JSON
  -n, --limit int      Number of events to show (default 20)
  -t, --type string    Filter by event type (e.g. agent.dreamed)
  -w, --world string   Filter by world ID
```
