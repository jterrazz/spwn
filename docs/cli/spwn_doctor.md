## spwn doctor

Check your spwn environment for issues

### Synopsis

Run diagnostic checks on your spwn environment.

Verifies Docker connectivity, images, configuration files, agents,
and authentication. Reports issues with suggested fixes.

```
spwn doctor [flags]
```
