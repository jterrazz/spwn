## spwn team new

Create a new team

```
spwn team new <name> [flags]
```

### Examples

```
  spwn team new "Matrix Ops" --color "#8B5CF6"
  spwn team new infra --color purple
```

### Options

```
      --color string         Accent color - hex (#8B5CF6) or name (purple, blue, red, amber, emerald)
      --description string   Short description
  -h, --help                 help for new
```
