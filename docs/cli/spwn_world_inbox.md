## spwn world inbox

Show messages in a world's inbox

### Synopsis

Show messages from the inbox inside a running world.

If an agent name is provided, shows only that agent's messages.
If not, shows all messages across all inboxes.

```
spwn world inbox <world-id> [agent-name] [flags]
```
