## spwn visitor

Spawn an ephemeral agent — no Mind, no memory, fire and forget

### Synopsis

Visitors are ephemeral agents that execute a single task and disappear.
They have no persistent identity, no Mind, and no journal.
Perfect for linting, testing, health checks, and one-off validations.

```
spwn visitor [task] [flags]
```
