## spwn skill

Author and manage reusable skill files

### Synopsis

Skills are procedures, playbooks, or pieces of knowledge - authored in markdown.

Attach one to an agent with:
  spwn agent add <agent> --skill <skill-name>

```
  -h, --help   help for skill
```
