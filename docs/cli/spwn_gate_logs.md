## spwn gate logs

Stream gate logs (docker logs)

```
spwn gate logs [flags]
```

### Options

```
  -f, --follow     Follow log output
  -h, --help       help for logs
  -n, --tail int   Number of lines from the tail (0 = all) (default 50)
```
