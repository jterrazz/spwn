## spwn team edit

Edit a team's metadata

```
spwn team edit <slug> [flags]
```

### Examples

```
  spwn team edit matrix-ops --color "#A855F7"
  spwn team edit infra --description "Infrastructure & ops"
```

### Options

```
      --color string         Accent color — hex or name
      --description string   Short description
  -h, --help                 help for edit
```
