## spwn team assign

Assign an agent to a team (or --clear to remove)

```
spwn team assign <agent-name> <team-slug> [flags]
```

### Examples

```
  spwn team assign neo matrix-ops
  spwn team assign qa --clear
```

### Options

```
      --clear   Remove agent from team
  -h, --help    help for assign
```
