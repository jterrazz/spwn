# Observatory

> Visual dashboard for monitoring universes, agents, and their evolution.

**Status:** Planned (not yet implemented)

## Vision

The Observatory is the visual interface for spwn — an isometric dashboard showing:
- Active universes and their physics
- Agent lifecycles, mind evolution, and journal history
- Gate bridges and element usage
- Real-time logs and metrics

## Stack (planned)

- Next.js + React
- Real-time WebSocket connection to spwn CLI/API
- Isometric 3D visualization (Three.js or similar)
