
⬡ auth - manage credentials

Usage:
  spwn auth [command]

Inspect:
  status                           Show every detected credential (default when no subcommand)
  check [provider]                 Validate active credentials against each API

Configure:
  use <provider> <method>          Pick oauth or api_key for a provider
  login <provider>                 Register credentials (--api-key <val> or --oauth)
  default <provider>               Prefer this provider when multiple are authenticated
  disable <provider>               Ignore a provider without deleting creds
  enable <provider>                Reverse a previous disable

Remove:
  logout <provider>                Clear all credentials for a provider (keychain, files)

Supported providers:
  anthropic                        Claude — OAuth (subscription) or API key
  openai                           Codex — OAuth (ChatGPT Plus) or API key (env-only today)
