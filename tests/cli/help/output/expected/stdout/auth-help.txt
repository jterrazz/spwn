
⬡ auth - manage credentials

Usage:
  spwn auth [command]

Commands:
  status                           Show authentication status (default when no subcommand)
  providers                        List configured AI providers
  login                            Set up credentials (Keychain or manual)
  logout                           Clear cached credentials
  token <token>                    Set a token directly (CI / scripts)
  check                            Validate credentials for all AI providers

Environment Variables:
  ANTHROPIC_API_KEY                Anthropic Claude API key
  OPENAI_API_KEY                   OpenAI API key
